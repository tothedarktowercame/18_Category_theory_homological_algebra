\subsection*{Synopsis}
Christensen shows that a set of objects in a complete cocomplete abelian category generates a projective class, and if these generators are small, the chain complex category admits a model structure reflecting that homological algebra. Motivated by pure derived categories.
\begin{enumerate}
  \item Describes projective classes generated by object sets.
  \item Constructs model structures on chain complexes reflecting given projective classes when generators are small.
  \item Applies to pure derived categories.
\end{enumerate}
