\subsection*{Synopsis}
Jonathan Davies promotes SÃ¶zer--Virelizier $\chi$-graded categories to the setting of an arbitrary group homomorphism $\tau\colon G\to H$, simultaneously grading objects and morphisms. A "half-enriched" Yoneda lemma controls representable $\tau$-graded functors, while semisimple $\tau$-graded categories admit block decompositions governed by the image of $\tau$. A pseudofunctor valued description in $\mathbf{Cat}$ packages $\tau$-graded categories as weak actions of $H$ with coherence inherited from $G$.
\begin{enumerate}
  \item Defines $\tau$-graded categories and their morphisms, extending degree constraints to objects.
  \item Establishes a Yoneda lemma tracking homogeneous components of enriched representables.
  \item Proves a structure theorem for semisimple $\tau$-graded categories and compatibly graded functors.
  \item Recasts the theory via pseudofunctors $BH \to \mathbf{Cat}$ endowed with $G$-indexed coherence data.
\end{enumerate}
