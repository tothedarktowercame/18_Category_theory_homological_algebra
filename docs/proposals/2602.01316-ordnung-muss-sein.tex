\subsection*{Synopsis}
Henning Krause proves that any length category with a partial order on simples satisfying explicit axioms is equivalent to finite-dimensional representations of that poset. The rules are necessary and sufficient, extending classical artinian representation theory to ordered settings.
\begin{enumerate}
  \item Formulates axioms on the partial order of simples in a length category.
  \item Shows equivalence with representations of the resulting poset category.
  \item Provides structural consequences for length categories satisfying the order rules.
\end{enumerate}
