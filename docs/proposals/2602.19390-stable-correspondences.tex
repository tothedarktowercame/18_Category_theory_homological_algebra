\subsection*{Synopsis}
The authors extend ambidexterity theorems using stable correspondences modeled by spans in symmetric monoidal $\infty$-categories, yielding new norm/trace formulas.
\begin{enumerate}
  \item define stable correspondences and the associated adjoint pairs,
  \item prove an ambidexterity theorem for spans in spectral categories,
  \item apply it to power operations on motivic spectra.
\end{enumerate}
