\subsection*{Synopsis}
The authors introduce "span logoi": logoi enriched by span correspondences designed for condensed mathematics. Contributions include:
\begin{enumerate}
  \item building span logoi from condensed sets and topological groups,
  \item relating them to Lurie's spectral logoi, and
  \item extracting symmetry data from span-based Galois actions.
\end{enumerate}
For PlanetMath we will ground these notions by:
\begin{itemize}
  \item mapping "span logoi" onto existing entries on logoi and condensed mathematics,
  \item detailing one example using condensed tori, and
  \item clarifying how span symmetry recovers classical Galois theory.
\end{itemize}
